🟩 SECTION 0 — Couleurs ANSI (Termux)
Utilisables dans tous tes scripts :

- Rouge : \e[31m
- Vert : \e[32m
- Jaune : \e[33m
- Bleu : \e[34m
- Magenta : \e[35m
- Cyan : \e[36m
- Reset : \e[0m

Exemple :
`bash
echo -e "\e[32m[OK]\e[0m Module chargé."
`
